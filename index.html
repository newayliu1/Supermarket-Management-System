    Create Instructions

  1. Look up the product which you want to create an order or inventory for in FIELD 1.
  2. If the product does not exists in databse, please create product in FIELD 2
  3. If the product exists in the database, use the product ID to create it in FIELD 3
Attention:
The category or section of product is the name of your inventory to store product.

    Update Instructions

  1. Look up the product which you want to update the inventory for from FIELD 1.
  2. Use the inventory id to update the price or section from FIELD 2.
Attention:
Order is not allowed to update!!

    Delete Instructions

  1. Look up the product which you want to delete the order and inventory for in FIELD 1.
  2. Use the inventory id or order id to delete in FIELD 2.

    Search Instructions

  1. Look up the product which you want to search the order and inventory for in FIELD 1.
  2. Use the inventory id or order id to Search in FIELD 2.
